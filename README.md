# quakus-graphql-demo Project

## Running the application in dev mode

    ./gradlew quarkusDev

## Running the test code in Python

    cd test/python
    python3 -m venv venv
    . venv/bin/activate
    pip install -r requirements.txt
    python3 subscriber.py

## Running the test code in NodeJS

    cd test/nodejs
    npm install
    node subscriber.js
